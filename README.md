# 🌸 Flower Classification using Machine Learning

This project is a machine learning–based flower classification system that predicts the flower category from an input image.

## 📌 Project Overview
- Image classification using deep learning
- Trained on a flower dataset
- Predicts flower type from user-provided images

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Matplotlib

## 📂 Project Structure

This structure separates training, testing, and inference data for clarity and reproducibility.
